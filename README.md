# EARIST-Extension-Services-System
EARIST Extension Service System (EESS) is a web-based platform for managing, tracking, and recommending extension programs at EARIST. It features analytics, user management, program lifecycle tracking, AI-powered recommendations, and a public interface for community engagement.
